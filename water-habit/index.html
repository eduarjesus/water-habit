Integración con Google Sheets
1. Abre script.google.com, crea un nuevo proyecto.
2. Pega el código del archivo google-apps-script.gs.
3. Despliega como Web App (acceso: Cualquiera).
4. Copia la URL y pégala abajo:

Atajos de iPhone

Configurar Atajo
1. Abre la app Atajos en tu iPhone.
2. Crea un nuevo Atajo → Obtener contenidos de URL.
3. URL: la misma URL de tu Google Apps Script.
4. Método: POST. Cuerpo JSON:
{"amount": 250, "source": "atajo"}
5. Agrega el Atajo a tu pantalla de inicio o como widget.

Tip: Crea uno por cada cantidad (250, 500, 750 ml) con íconos de vasos para acceso rápido. El mismo script de Google procesará los datos.

Instalar como app

Guardar en pantalla de inicio
iPhone: Safari → botón compartir ↑ → "Agregar a pantalla de inicio".

Android: Chrome → menú ⋮ → "Agregar a pantalla de inicio".

La app funciona sin conexión gracias al Service Worker incluido.
